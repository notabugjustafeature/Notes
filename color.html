There are 3 ways to do color
- name
- hex
- styles using rgba

name is limited to 140 defined names that you can lookup

If you use hex, you have 16,777,216
- syntax: you are going to use a # in front of the color - web safe colors use repeated values

RGBA which stands for: Red, Green, Blue, Alpha
- can only be used with stylesheets
- Syntax will be rgba(0-255, 0-255, 0-255, 0-1) (the alpha is assumed 1)
